## mercury_gve

Парсер личного кабинета для личного пользования

## Стак
python 3.10
poetry

- aiogram
- aioschedule
- beautifulsoup

### подготовка
`default.settings.py` переименовать в `settings.py` и отредактировать

установить зависимости `python -m pip install -r requirements.txt`

### запуск
`python merc_gve/main.py`
